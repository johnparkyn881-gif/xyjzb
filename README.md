# 校园记账本

一个专为大学生设计的简洁记账工具，帮助你轻松管理生活费，养成理财好习惯。

## ✨ 功能特性

- 💰 **用户系统**: 注册登录，数据独立存储
- 📝 **账单管理**: 快速记录收入和支出
- 📊 **数据统计**: 多维度数据分析和可视化
- 🎨 **现代设计**: 渐变色彩、流畅动画、深色模式
- 📱 **响应式**: 完美支持手机、平板、桌面设备
- 💾 **数据管理**: 支持数据导入导出

## 💻 本地运行

1. 克隆仓库
```bash
git clone https://github.com/你的用户名/budget-tracker.git
cd budget-tracker
```

2. 打开网站
```bash
# 方法1: 直接在浏览器中打开 index.html
open index.html

# 方法2: 使用本地服务器（推荐）
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 📦 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **数据存储**: LocalStorage
- **图表库**: Chart.js 4.4.1
- **字体**: Google Fonts (Inter)

## 📖 使用说明

1. **注册账号**: 首次使用需要注册一个账号
2. **记账**: 点击"记一笔"按钮添加收入或支出
3. **查看统计**: 在"统计分析"页面查看详细的数据分析
4. **管理账单**: 在"账单管理"页面编辑或删除记录
5. **导出数据**: 在"设置"页面导出备份数据
